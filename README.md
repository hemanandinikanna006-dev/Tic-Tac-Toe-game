# Tic-Tac-Toe-game

###  [Play the Game Here!](https://hemanandinikanna006-dev.github.io/Tic-Tac-Toe-game/)

A modern, creative take on the classic Tic-Tac-Toe game. Built with HTML5, CSS3, and JavaScript, this project features a vibrant "Neon Glow" aesthetic set against a custom photographic background.

# Features
Creative UI: Features a "Glassmorphism" design with semi-transparent game tiles.
Neon Aesthetics: Dynamic glowing effects for 'X' (Cyan) and 'O' (Purple) symbols.
Responsive Logic: Accurate win detection for all 8 possible patterns.
Custom Background: Optimized for a high-quality visual experience with a dark overlay to make game elements pop.
Smooth Reset: Fully functional reset system that clears the board and allows for instant replay.

## How to Play
The Goal: Be the first player to get 3 of your marks in a row (up, down, across, or diagonally).
Turns: Player 'X' starts first, followed by Player 'O'.
Winning: Once a winner is detected, a congratulatory message appears, and the board locks to show the winning pattern.
Restart: Click the Reset Game button at any time to clear the board and start a fresh round.

## Technologies Used
HTML5: Structured the game board and layout.
CSS3: Implemented text-shadow glows, backdrop-filter for glass effects, and background-attachment: fixed for a stable UI.
JavaScript: Managed game logic, event listeners, and win-pattern verification.
